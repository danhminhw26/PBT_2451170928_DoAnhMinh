Câu A1:
type="email" → Ô nhập text, kiểm tra có dấu @ và định dạng email hợp lệ → Dùng cho đăng ký / đăng nhập tài khoản
type="password" → Ô nhập text nhưng ký tự bị ẩn (•••••) → Dùng cho nhập mật khẩu khi đăng nhập / thanh toán
type="text" → Ô nhập văn bản bình thường → Dùng nhập tên khách hàng, địa chỉ
type="number" → Ô nhập số, có nút tăng/giảm → Kiểm tra chỉ nhập số → Dùng chọn số lượng sản phẩm
type="tel" → Ô nhập số điện thoại → Không validate chặt nhưng hỗ trợ bàn phím số trên mobile → Dùng nhập số điện thoại giao hàng
type="url" → Ô nhập link, kiểm tra định dạng URL → Dùng nhập website (ví dụ shop đăng ký bán hàng)
type="date" → Hiển thị lịch chọn ngày → Kiểm tra định dạng ngày → Dùng chọn ngày giao hàng
type="radio" → Nút chọn 1 trong nhiều lựa chọn → Dùng chọn phương thức thanh toán (COD, chuyển khoản)
type="checkbox" → Ô tick nhiều lựa chọn → Dùng chọn sản phẩm hoặc đồng ý điều khoản
type="file" → Nút upload file → Có thể giới hạn loại file → Dùng upload ảnh sản phẩm / avatar người dùng
